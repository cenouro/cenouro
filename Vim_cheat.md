# Motion
* `lkjh` - arrow keys
* `wW` - words
* `eE` - words ends
* `bB` - previous words

# Operations
* `~` - toggle case
* `Ctrl-D Ctrl-T` - same as `<< >>`, but in `insert mode`
* `Ctrl-F` - same as `==`, but in `insert mode`
* `.` - repeat last change
* `@:` - repeat last command-line

# Spell check
* `]s [s` - move to misspelled words
* `z=` - suggestions
